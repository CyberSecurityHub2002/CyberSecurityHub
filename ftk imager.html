FTK Imager Forensic Analysis

A walkthrough on forensic imaging and analysis using FTK Imager

🔗 GitHub Repository

Project repo: FTK-Imager-and-Forensic-Imaging-Analysis

📌 Overview

This project demonstrates how to use FTK Imager to perform disk acquisition and analyze evidence for digital forensics. It's part of a broader initiative to explore tools used in incident response and evidence handling.

📁 Contents

  • Installation and Setup of FTK Imager
  • Creating a Forensic Image
  • Hash Verification
  • Mounting the Image for Analysis
  • Live Memory Acquisition
  • Report Generation

🛠 Tools Used

  • FTK Imager
  • Windows OS (target system)
  • Hashing tools (e.g., SHA256sum)
